# alzheimer-forecast
#Прогнозирование распространённости болезни Альцгеймера с помощью Prophet и ARIMA

## Автор
Мозоль Инга Евгеньевна, СПбГУ

## Описание проекта
Данный проект выполнен в рамках научно-исследовательской работы по теме «Анализ и прогнозирование временных рядов распространённости неинфекционных заболеваний на примере болезни Альцгеймера с использованием алгоритмов машинного обучения».

## Цель
Построение и исследование моделей прогнозирования временных рядов распространённости болезни Альцгеймера с использованием Prophet и ARIMA.

## Структура репозитория
- `data/` – исходные данные (Data.xlsx)
- `notebooks/` – Jupyter Notebook с кодом для двух семестров
- `results/` – результаты экспериментов (CSV-файлы)
- `figures/` – графики прогнозов по странам

## Используемые модели
- Prophet (Meta/Facebook)
- ARIMA (AutoRegressive Integrated Moving Average)

## Страны
Австрия, Бельгия, Германия, Греция, Италия, Нидерланды, Португалия, Сингапур, Финляндия, Швейцария, Швеция

## Период
1990–2021 (исторические данные), прогноз до 2031 года

## Результаты
- Prophet применим ко всем 11 странам
- ARIMA работоспособна только для 5 стран
- Наибольшая прогнозируемая нагрузка в 2031 году – Германия (9,06 млн человек)
